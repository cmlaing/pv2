WebApp
A Social Networking Example

The purpose of the project was to create a web application that demonstrated the use of chart widgets and localStorage using JavaScript. CSS Grid and Flexbox were used primarily for the layout.

JavaScript: chart widgets, an alert notifcation box, green icon notification, localStorage, message sent confirmation.

The top-right corner of the page has a bell icon with a green dot. The green dot indicates there are new notifications for the user. When the bell is clicked, a drop down menu appears of the notifications and the green dot disappears.
Further down the page is a purple alert banner. When the "x" is clicked, the banner disappears.
When the user attempts to send a message, all fields (selected user, message) must be filled out or else an error message (with instructions) will appear. If successful, a confirmation message appears on screen for the user.
The settings section uses localStorage to save the settings even after the page is refreshed. The cancel button negates any changes and resets to base values. The settings that are able to be saved include two toggle switches (on/off) and a drop down menu to select a timezone.

Overall, this project demonstrates a modular approach of having multiple sections come together in a coherent design. JavaScript plugins keep the chart information up to date without having to manually update the webpage itself.
